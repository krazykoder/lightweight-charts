Basic Shapes

Demonstrates creating a Shape Series with standard shapes.

const chart = LightweightCharts.createChart(container, { width: 600, height: 300 });
const series = chart.addShapeSeries({
    position: 'top',
    shape: 'arrowDown',
    color: '#E91E63',
    size: 15,
});
// Data points with default shape
series.setData(data);

Per-Point Customization

Override shape, color, and size on individual data points.

const series = chart.addShapeSeries({ position: 'top', shape: 'circle' });
series.setData([
    { time: '2019-04-11', value: 0, shape: 'square', color: 'red' },
    { time: '2019-04-12', value: 0, shape: 'diamond', size: 20 },
    // ...
]);

Text Labels

Display text labels alongside shapes.

const series = chart.addShapeSeries({ 
    position: 'bottom', 
    shape: 'arrowUp',
    labelOffset: 10 
});
series.setData([
    { time: '2019-04-11', value: 0, text: 'Buy Signal' },
    // ...
]);
