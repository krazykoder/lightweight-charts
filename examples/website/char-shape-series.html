Emojis

Demonstrates creating a Char Shape Series with emojis.

const chart = LightweightCharts.createChart(container, { width: 600, height: 300 });
const series = chart.addCharShapeSeries({
    position: 'top',
    size: 20,
});
// Data points with emojis
series.setData([
    { time: '2019-04-11', value: 0, char: '🚀' },
    { time: '2019-04-12', value: 0, char: '💰' },
]);

Letters & Numbers

Using alphanumeric characters as markers.

const series = chart.addCharShapeSeries({ position: 'top', size: 16 });
series.setData([
    { time: '2019-04-11', value: 0, char: 'A', color: 'red' },
    { time: '2019-04-12', value: 0, char: 'B', color: 'blue' },
    { time: '2019-04-13', value: 0, char: '1', color: 'green' },
]);

Text Labels

Display text labels alongside chars.

const series = chart.addCharShapeSeries({ 
    position: 'bottom', 
    size: 20,
    labelOffset: 10 
});
series.setData([
    { time: '2019-04-11', value: 0, char: '☀️', text: 'Sunny' },
    // ...
]);
