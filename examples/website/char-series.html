Emojis & Currency

Display rich text characters including standard emojis and currency symbols.

const series = chart.addCharSeries({
    size: 24,
});
series.setData([
    { time: '2019-04-11', value: 80, char: '🚀' },
    { time: '2019-04-12', value: 82, char: '€' },
    { time: '2019-04-13', value: 85, char: '₿' },
]);

Per-Point Customization

Override char, color, and size on individual data points.

const series = chart.addCharSeries({ size: 14, color: '#333' });
series.setData([
    { time: '2019-04-11', value: 80, char: 'A', color: 'red', size: 30 },
    { time: '2019-04-12', value: 82, char: 'B' }, // Default style
    { time: '2019-04-13', value: 81, char: 'C', color: 'blue' },
]);

Vertical Offset Positioning

Use positive offset to position characters above the price point, negative to position below.

const series = chart.addCharSeries({
    size: 20,
    color: '#333',
    offset: 15, // Characters appear 15px above the price
});
series.setData([
    { time: '2019-04-11', value: 80, char: '▲' },
    { time: '2019-04-12', value: 82, char: '▼' },
    { time: '2019-04-13', value: 85, char: '■' },
]);
