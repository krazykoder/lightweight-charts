Classic Price-Volume Chart

Demonstrates combining an area series with a volume histogram using separate price scales.

const chart = LightweightCharts.createChart(container, {
    width: 700,
    height: 300,
    rightPriceScale: {
        scaleMargins: { top: 0.3, bottom: 0.25 },
    },
});

const areaSeries = chart.addAreaSeries({
    topColor: 'rgba(38,198,218, 0.56)',
    bottomColor: 'rgba(38,198,218, 0.04)',
    lineColor: 'rgba(38,198,218, 1)',
    lineWidth: 2,
});

const volumeSeries = chart.addHistogramSeries({
    color: '#26a69a',
    priceFormat: { type: 'volume' },
    priceScaleId: '', // Unset the price scale ID
});

chart.priceScale('').applyOptions({
    scaleMargins: { top: 0.8, bottom: 0 },
});

Candlestick with Squeeze Histogram

Candlestick chart overlaid with color-coded squeeze indicator histogram on separate scales.

const candlestickSeries = chart.addCandlestickSeries({
    upColor: '#26a69a',
    downColor: '#ef5350',
});

const sqzHistogram = chart.addHistogramSeries({
    priceScaleId: '',  // Use separate price scale
});

chart.priceScale('').applyOptions({
    scaleMargins: { top: 0.7, bottom: 0 },
});

const sqzColors = { 1: '#26a69a', 2: '#ef5350', 3: '#64b5f6', 4: '#ffc107' };
const sqzData = data.map(d => ({
    time: d.time,
    value: d.SQZ_Hist || 0,
    color: sqzColors[d.SQZ_HistC] || 'gray',
}));
sqzHistogram.setData(sqzData);

Candlestick with Volume and Squeeze Data

Advanced example showing three separate scales: candlesticks (middle), volume (bottom), and squeeze histogram (top).

const chart = LightweightCharts.createChart(container, {
    rightPriceScale: {
        scaleMargins: { top: 0.25, bottom: 0.35 },
    },
});

const candlestickSeries = chart.addCandlestickSeries({
    upColor: '#26a69a',
    downColor: '#ef5350',
});

// Volume at the bottom (priceScaleId: 'volume')
const volumeSeries = chart.addHistogramSeries({
    priceFormat: { type: 'volume' },
    priceScaleId: 'volume',
});
chart.priceScale('volume').applyOptions({
    scaleMargins: { top: 0.9, bottom: 0 },
});

// Squeeze Data at the top (priceScaleId: '')
const sqzHistogram = chart.addHistogramSeries({
    priceScaleId: 'sqzHistogram',
});
chart.priceScale('sqzHistogram').applyOptions({
    scaleMargins: { top: 0, bottom: 0.75 },
});
