Two Price Scales

This example demonstrates two series using separate price scales (Left and Right). Useful for comparing data with different units or magnitudes.

const chart = createChart(container, {
    leftPriceScale: {
        visible: true,
        borderColor: '#555ffd',
    },
    rightPriceScale: {
        visible: true,
        borderColor: '#26a69a',
    },
});

const candleSeries = chart.addCandlestickSeries({
    priceScaleId: 'left', // Bind to Left Scale
    upColor: '#555ffd', 
    downColor: '#555ffd',
});

const lineSeries = chart.addLineSeries({
    color: '#26a69a',
    priceScaleId: 'right', // Bind to Right Scale
    lineWidth: 2,
});

candleSeries.setData(data1);

Inverted Scale

Invert the price scale to show inverse relationships. In this example, the right scale is inverted.

const chart = createChart(container, {
    rightPriceScale: {
        visible: true,
        invertScale: true, // Invert the scale
    },
});

const areaSeries = chart.addAreaSeries({
    topColor: 'rgba(239, 83, 80, 0.4)',
    bottomColor: 'rgba(239, 83, 80, 0.05)',
    lineColor: '#ef5350',
});

No Overlap (Margins)

Use scaleMargins to prevent series from overlapping. Here, volume is pinned to the bottom 20%, and price takes the top 80%.

const chart = createChart(container, {
    rightPriceScale: {
        scaleMargins: {
            top: 0.1,
            bottom: 0.3, // Leave space at bottom for volume
        },
    },
});

const priceSeries = chart.addCandlestickSeries();

// Create a separate scale for volume to manage its margins independently
const volumeSeries = chart.addHistogramSeries({
    priceScaleId: 'vol', // Custom ID
    priceFormat: { type: 'volume' },
});

chart.priceScale('vol').applyOptions({
    scaleMargins: {
        top: 0.8, // Push volume to bottom 20%
        bottom: 0,
    },
});
