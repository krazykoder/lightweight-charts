Single Line Legend

Classic value display with a custom color-coded title.

chart.subscribeCrosshairMove(param => {
    if (!param.time) {
        updateLegend(lastData);
        return;
    }
    const data = param.seriesData.get(lineSeries);
    legend.innerHTML = `<span>Price:</span> ${data.value}`;
});
