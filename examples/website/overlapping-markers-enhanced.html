Basic Overlapping Markers

Multiple markers at the same time with different shapes and colors.

const series = chart.addCandlestickSeries();
series.setData(data);

const markers = [
  { time: data[30].time, position: 'belowBar', 
    color: 'rgba(213, 0, 0, 1)', shape: 'circle' },
  { time: data[30].time, position: 'belowBar', 
    color: 'rgba(25, 118, 210, 1)', shape: 'arrowUp' },
  { time: data[50].time, position: 'aboveBar', 
    color: 'rgba(25, 118, 210, 1)', shape: 'circle' },
  { time: data[50].time, position: 'aboveBar', 
    color: 'rgba(213, 0, 0, 1)', shape: 'arrowDown' }
];
series.setMarkers(markers.sort((a, b) => a.time - b.time));
