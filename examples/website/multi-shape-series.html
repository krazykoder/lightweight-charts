Simple Stacking

Two stacked series at different levels (Top & Bottom).

chart.addShapeSeries({ position: 'top', level: 0, shape: 'circle' });
chart.addShapeSeries({ position: 'top', level: 1, shape: 'square' });

Shapes on Price Levels

Using 'value' position with fixedValue to place shapes at specific prices.

// Series at value 20
const s1 = chart.addShapeSeries({ 
    position: 'value', 
    fixedValue: 20, 
    shape: 'arrowUp', 
    color: 'red' 
});
s1.setData([
    { time: '2018-12-19', value: 0, text: '20' },
    ...
]);

// Series at value 15
const s2 = chart.addShapeSeries({ 
    position: 'value', 
    fixedValue: 15, 
    shape: 'arrowDown', 
    color: 'blue' 
});
s2.setData([
    { time: '2018-12-19', value: 0, text: '15' },
    ...
]);
