Baseline Series Examples

Baseline with Fixed Price Base

This chart sets a fixed base value at price 40. Values above 40 are green (top), and values below are red (bottom).

const chart = LightweightCharts.createChart(document.getElementById('chart-container1'), { width: 600, height: 300 });
const baselineSeries = chart.addBaselineSeries({
    baseValue: { type: 'price', price: 40 },
    topLineColor: 'rgba(38, 166, 154, 1)',
    topFillColor1: 'rgba(38, 166, 154, 0.28)',
    topFillColor2: 'rgba(38, 166, 154, 0.05)',
    bottomLineColor: 'rgba(239, 83, 80, 1)',
    bottomFillColor1: 'rgba(239, 83, 80, 0.05)',
    bottomFillColor2: 'rgba(239, 83, 80, 0.28)',
});
baselineSeries.setData(data);

Baseline with Custom Style

This chart uses a different base value and custom colors for a distinct look.

const chart2 = LightweightCharts.createChart(document.getElementById('chart-container2'), { width: 600, height: 300 });
const baselineSeries2 = chart2.addBaselineSeries({
    baseValue: { type: 'price', price: 0 },
    topLineColor: '#2962FF',
    topFillColor1: 'rgba(41, 98, 255, 0.28)',
    topFillColor2: 'rgba(41, 98, 255, 0.05)',
    bottomLineColor: '#FF6D00',
    bottomFillColor1: 'rgba(255, 109, 0, 0.05)',
    bottomFillColor2: 'rgba(255, 109, 0, 0.28)',
    lineWidth: 4,
});
baselineSeries2.setData(data);
