Line Series with Vertical Lines

Demonstrates sticky vs. non-sticky axis labels.

const chart = LightweightCharts.createChart(document.getElementById('chart-container1'), {
    width: 600,
    height: 300,
});
const series = chart.addLineSeries({
    lineType: LightweightCharts.LineType.WithSteps,
});
series.setData(data);

// Non-sticky label (default)
const vLine1 = series.createVerticalLine({
    time: data[50].time,
    color: '#EF5350',
    title: 'Event A',
    // axisLabelVisibleAtEdge: false, // default
});

// Sticky label
series.createVerticalLine({
    time: data[100].time,
    color: '#26a69a',
    title: 'Start',
    axisLabelVisibleAtEdge: true,
});

Vertical Lines on Histogram (Area Style)

Vertical lines can be added to any series type, including Histograms.

const chart2 = LightweightCharts.createChart(document.getElementById('chart-container2'), {
    width: 600,
    height: 300,
});
const histogramSeries = chart2.addHistogramSeries({
    histogramStyle: 1, // Area
});
histogramSeries.setData(histogramData);

histogramSeries.createVerticalLine({
    time: histogramData[10].time,
    color: '#fff',
    title: 'Hist Event 1',
    axisLabelVisibleAtEdge: true,
});
